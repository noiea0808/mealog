# mealog PWA 설치 가이드

## 📱 앱으로 설치하기

mealog을 PWA(Progressive Web App)로 설치하면 일반 앱처럼 사용할 수 있습니다!

### 🖥️ 데스크톱 (Windows/Mac/Linux)

**Chrome/Edge:**
1. 웹사이트 접속: `http://localhost:8000`
2. 주소창 오른쪽의 **설치 아이콘** 클릭
3. 또는 메뉴 > "앱 설치" 선택
4. "설치" 버튼 클릭

**결과:**
- 바탕화면에 아이콘 생성
- 시작 메뉴에 추가
- 독립 창으로 실행

### 📱 모바일 (Android)

**Chrome:**
1. 웹사이트 접속
2. 메뉴(⋮) > "홈 화면에 추가" 선택
3. 또는 주소창 아래 "홈 화면에 추가" 팝업 확인

**결과:**
- 홈 화면에 아이콘 추가
- 일반 앱처럼 실행
- 전체 화면 모드

### 📱 모바일 (iOS - Safari)

1. Safari로 웹사이트 접속
2. 공유 버튼(□↑) 클릭
3. "홈 화면에 추가" 선택
4. 이름 확인 후 "추가" 클릭

**결과:**
- 홈 화면에 아이콘 추가
- 독립 앱처럼 실행

## 🎨 아이콘 생성하기

PWA를 제대로 사용하려면 아이콘이 필요합니다:

1. `create-icons.html` 파일을 브라우저에서 열기
2. "아이콘 생성" 버튼 클릭
3. `icon-192.png`와 `icon-512.png` 파일이 다운로드됨
4. 다운로드된 파일을 `mealog` 폴더에 복사

## ✅ 체크리스트

PWA 설치 전 확인:
- [ ] `manifest.json` 파일이 있나요?
- [ ] `sw.js` 파일이 있나요?
- [ ] `icon-192.png`와 `icon-512.png` 파일이 있나요?
- [ ] 웹 서버를 통해 접속하고 있나요? (HTTPS 또는 localhost)

## 🔧 문제 해결

**"설치" 버튼이 안 보여요:**
- HTTPS 또는 localhost로 접속해야 합니다
- `file://` 프로토콜로는 작동하지 않습니다
- 웹 서버를 통해 접속하세요

**아이콘이 안 보여요:**
- `icon-192.png`와 `icon-512.png` 파일이 루트 폴더에 있는지 확인
- `create-icons.html`로 아이콘을 생성하세요

**Service Worker 오류:**
- HTTPS 또는 localhost에서만 작동합니다
- `sw.js` 파일이 루트 폴더에 있는지 확인

## 📝 참고사항

- PWA는 오프라인에서도 일부 기능을 사용할 수 있습니다
- 데이터는 Firebase에 저장되므로 인터넷 연결이 필요합니다
- 설치 후에도 일반 브라우저처럼 업데이트됩니다
