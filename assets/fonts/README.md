# 앨범 캡션용 폰트 (나눔손글씨 가람연꽃)

앨범 사진 아래 "메뉴 @ 장소" 캡션에 **나눔손글씨 가람연꽃** 폰트를 적용하려면 아래 파일을 이 폴더에 넣어주세요.

- `NanumGaramYeonkot.woff2` 또는 `NanumGaramYeonkot.woff`

다운로드: [눈누 - 가람연꽃](https://noonnu.cc/font_page/605) 또는 [네이버 나눔글꼴](https://hangeul.naver.com/)  
(woff/woff2 변환은 온라인 변환 도구 사용 가능)

파일을 넣지 않으면 비슷한 손글씨체인 **Nanum Pen Script**가 적용됩니다.
